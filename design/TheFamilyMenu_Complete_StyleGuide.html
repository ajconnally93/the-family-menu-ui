Complete Style Guide for The Family Menu

This style guide defines the visual identity for a family-focused meal planning web application. The goal is to make the interface feel warm, organized, practical, and easy to use for busy households.

Since the application helps users plan meals, build grocery lists, and manage weekly food decisions, the design should feel calm and trustworthy rather than flashy or overly trendy.

Primary Action Secondary Action Outline Button
Brand personality

Simple. Family-oriented. Efficient.

  • Warm, clean colors inspired by fresh ingredients and home cooking
  • Readable typography for busy users scanning recipes and grocery lists
  • Card-based layouts that work naturally with Bootstrap grid classes
  • Clear calls to action for adding meals, viewing plans, and checking grocery lists

1. Brand Direction

Visual tone

The interface should feel welcoming and practical. Users are often planning meals while distracted, tired, or multitasking, so the layout should reduce cognitive load and emphasize clarity.

Warm Organized Friendly Reliable Minimal Approachable

Bootstrap implementation notes

Use Bootstrap for layout, forms, buttons, spacing, and responsive behavior. Apply custom classes on top of Bootstrap components to preserve a consistent brand look.

Use Bootstrap forGrid, containers, navbars, forms, modals, spacing utilities, responsive layout
CustomizeColors, border radius, shadows, button styling, card presentation, heading typography
AvoidOverusing saturated default Bootstrap blues if they clash with the warm home-cooking aesthetic

9. Spacing, Layout, and Responsiveness

Do

Use generous whitespace, consistent card padding, and clean grouping between recipe and grocery content.

Don’t

Do not overcrowd screens with too many competing actions, dense text blocks, or inconsistent spacing.

10. Bootstrap Mapping

LayoutUse .container, .row, and .col-* for page structure
ButtonsStart from Bootstrap button classes and override colors, border radius, and shadows
FormsUse Bootstrap form controls, then apply custom brand focus states
CardsBootstrap cards are a good base, but increase border radius and soften colors
NavbarUse Bootstrap navbar structure, but keep link count low and styling warm/minimal
Final note: This style guide is designed to support the app’s main promise: helping busy users move from choosing meals to building a complete grocery list with as little friction as possible.
